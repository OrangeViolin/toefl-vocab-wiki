# nominal

> **发音**：[ˈnɑːmɪnl]　|　**所在**：[[Word-List-2]]（WL02-017）

## 释义

- a. 名义上的；（费用等）微不足道的

## 记忆方法

词根记忆：nomin（名称）＋al→名义上的

## 常见搭配

- nominal assets 名义资产

## 例句

- Frank is only a nominal chairman; the real work is done by others. 弗兰克只是名义上的主席，实际工作是其他人做的。
