# omit

> **发音**：[əˈmɪt]　|　**所在**：[[Word-List-38]]（WL38-020）

## 释义

- vt. 省略，删去；遗漏，忽略（neglect）

## 记忆方法

联想记忆：om（音似：“呕”）＋it（它）→把它呕出去→省略

## 常见搭配

- omit doing/to do 忘记…

## 例句

- You'd better check the name list again to make sure there is no one omitted from it. 你最好再查看一下名单，确保没有遗漏任何人。
