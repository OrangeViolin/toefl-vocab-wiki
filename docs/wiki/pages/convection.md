# convection

> **发音**：[kənˈvekʃn]　|　**所在**：[[Word-List-36]]（WL36-038）

## 释义

- n. 对流；传送（conveying, transmission）

## 记忆方法

词根记忆：con（共同）＋vect（搬运）＋ion→一起搬运→对流

## 例句

- A heater is using convection when it warms the air in a room. 加热器给屋子里的空气加热时利用的是对流原理。
