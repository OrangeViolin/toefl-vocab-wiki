# core

> **发音**：[kɔːr]　|　**所在**：[[Word-List-31]]（WL31-058）

## 释义

- n. 地核；果心，核心（center）；要点，精髓 vt. 去掉…的中心部分

## 记忆方法

词根记忆：cor（心）＋e→核心

## 常见搭配

- to the core 十足，透顶

## 例句

- Concern for the disadvantaged groups is at the core of the argument. 对弱势群体的关注是争论的核心。
