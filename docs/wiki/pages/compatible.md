# compatible

> **发音**：[kəmˈpætəbl]　|　**所在**：[[Word-List-23]]（WL23-040）

## 释义

- a. 协调的，一致的（harmonious, coherent）；兼容的；能和睦相处的

## 记忆方法

词根记忆：com（共同）＋pat（＝path 感觉）＋ible→有共同感觉的→协调的

## 例句

- This project is not compatible with the company's long-term plans. 这个方案与公司的长远计划不一致。
