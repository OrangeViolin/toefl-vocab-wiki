# trace

> **发音**：[treɪs]　|　**所在**：[[Word-List-33]]（WL33-055）

- vt. 查出，找到；追溯；探索（explore）；描摹（describe）；追述 n. 痕迹（＊ imprint）；微量，少许；扫描线
