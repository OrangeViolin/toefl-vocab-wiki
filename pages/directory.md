# directory

> **发音**：[dəˈrektəri]　|　**所在**：[[Word-List-36]]（WL36-034）

## 释义

- n. 人名地址录，电话号码簿；目录

## 记忆方法

联想记忆：direct（指引）＋ory（表物）→指引人们查询的东西→人名地址录

## 常见搭配

- a telephone directory 电话号码簿

## 例句
