俞敏洪《TOEFL词汇词根＋联想记忆法（乱序版）》互动知识库。

遵循 [Memex Wiki 宪法](~/memex/CONSTITUTION.md)。

## 关于本 Wiki

基于新东方绿宝书系列——俞敏洪《TOEFL词汇词根＋联想记忆法（乱序版）》（西安交通大学出版社，2012）构建的结构化词汇知识库，涵盖：

- **单词**（4500+）：TOEFL 核心词汇，含发音、释义、例句、记忆法
- **词根**（200+）：拉丁/希腊词根，附带同根词家族
- **词缀**（100+）：前缀/后缀，含含义和例词
- **Word List**（48）：按原书乱序编排的章节

每个词条均标注 Word List 编号，可追溯至原文具体位置。
